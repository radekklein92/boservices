# Zvuky ohňostroje (volitelné)

Oslavný ohňostroj na Dashboardu (`FireworksCelebration`) má tlačítko pro zapnutí
zvuku. Zvuk se přehraje, pokud zde leží tyto soubory (jinak je ohňostroj tichý -
vizuál funguje vždy):

- `firework-1.mp3`
- `firework-2.mp3`
- `firework-3.mp3`

Stačí krátké (1-2 s) decentní zvuky výbuchu/praskání ohňostroje (doporučeno CC0 /
royalty-free, např. z Pixabay nebo Mixkit). Hlasitost je v komponentě nastavená
nízko (`volume 6-16`).
